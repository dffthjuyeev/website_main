2. Données collectées

Via le formulaire de contact de notre site, nous collectons les données suivantes :

  • Nom complet
  • Adresse email
  • Numéro de téléphone (optionnel)
  • Type de projet
  • Contenu de votre message

3. Finalité du traitement

Ces données sont collectées uniquement pour :

  • Répondre à vos demandes de devis et d'informations
  • Vous recontacter concernant votre projet

4. Base légale

Le traitement de vos données est fondé sur votre consentement, exprimé par l'envoi volontaire du formulaire de contact.

5. Destinataires des données

Vos données personnelles sont uniquement accessibles à GN Import & Export. Elles ne sont jamais vendues, échangées ou partagées avec des tiers, sauf obligation légale.

Le formulaire de contact utilise le service Formspree pour la transmission des messages. Consultez leur politique de confidentialité.

6. Durée de conservation

Vos données sont conservées pendant une durée maximale de 3 ans après le dernier contact, puis supprimées.

7. Vos droits

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger vos données inexactes
  • Droit de suppression : demander l'effacement de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit d'opposition : vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : gn-import-export@gmail.com

8. Cookies

Ce site n'utilise pas de cookies de tracking ni de cookies publicitaires. Aucune donnée de navigation n'est collectée à des fins statistiques ou commerciales.
